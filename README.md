# Power BI Project: Monthly Banking Transactions & Balances Dashboard

## Overview
This Power BI dashboard provides a detailed view of monthly banking transactions and balances for the year 2024. 
It helps users analyze transaction trends, balances, and performance across various cities, currencies, and customer demographics.

---

## Key Features
- **Interactive Filters:** Users can filter data by:
  - Bank Name (Sent & Received)
  - City
  - Device Type
  - Gender
  - Age Group
  - Merchant Name
  - Payment Method
  - Purpose
  - Transaction Type

- **Dynamic Visuals:** The dashboard contains interactive visualizations that switch between line and column charts for comparative analysis.

---

## Visuals Description

### 1. Balance by Month (Line Chart)
- Shows the **monthly balance trend** for the year 2024.
- Highlights peaks in **June (8.536K)** and steady performance in the remaining months.
- Interactive toggle allows switching between **line and column views**.

### 2. Transactions by Month (Line Chart)
- Displays **transaction volume trends** by month.
- The highest transaction volume is seen in **May (1.707K)**.
- Helps identify patterns in transaction activity.

### 3. Transactions by Month (Column Chart)
- Represents monthly **transaction amounts** in a bar chart format.
- Maintains consistency in values around **1.6M to 1.7M** throughout the year.

### 4. Balance by Month (Column Chart)
- Displays monthly **average balances** using column visuals.
- Values remain fairly stable between **8.2M and 8.5M**, suggesting consistent financial activity.

### 5. City-Wise & Currency Breakdown
- Compares cities: **Bangalore (EUR)**, **Delhi (USD)**, **Hyderabad (GBP)**, and **Mumbai (INR)**.
- Shows both **transaction amount** and **remaining balance** per city and month.
- Supports localized analysis for multi-currency banking data.

---

## Insights & Observations
- **June** recorded the highest balance and transaction activity.
- **Balances** remain stable across months, indicating consistent account management.
- **Transaction peaks** suggest periodic financial activity (likely payroll or billing cycles).
- **City-level comparison** helps identify regions with higher transaction volumes or remaining balances.
